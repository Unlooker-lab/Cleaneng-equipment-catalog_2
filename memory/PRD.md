# AVDIRVD - Каталог оборудования для автомоек

## Проблема
Компания AVDIRVD (avdirvd.by) продаёт профессиональное клининговое оборудование для автомоек. Требовался прототип улучшенной главной страницы каталога для WordPress-магазина.

## Архитектура
- **Frontend**: React + Tailwind CSS (прототип для адаптации в WordPress/Elementor)
- **Backend**: FastAPI + MongoDB (API каталога с фильтрацией)
- **Шрифт**: Manrope (Google Fonts, кириллица)
- **Цветовая схема**: #1D5EBC (blue primary), #0F172A (dark), #F8FAFC (subtle bg)

## Пользователи
- B2B покупатели: владельцы автомоек, операторы моек самообслуживания, промышленные компании (Беларусь/СНГ)

## Что реализовано (Январь 2026)
- Хедер: логотип, навигация, поиск, телефон, кнопка "Оставить заявку"
- Хлебные крошки
- Левый сайдбар: дерево категорий (16 категорий), фильтры по цене, бренду (7), давлению, потоку
- Сетка товаров (19 товаров из реального каталога + дополнительные)
- 3 режима просмотра: сетка, компакт, список
- Сортировка (6 вариантов)
- Активные теги фильтров с возможностью удаления
- Пагинация
- Секция "Новинки каталога"
- Секция "Наши партнёры" (TOR, Annovi Reverberi, HAWK, RM Suttner, Christ AG, CTW Cleaning, Groninger)
- Форма обратной связи (модальное окно) с сохранением в MongoDB
- Футер с контактами, навигацией
- Мобильная адаптация (гамбургер-меню, боковая панель фильтров)

## Тестирование
- Backend: 17/17 тестов (100%)
- Frontend: все функции протестированы (100%)

## Бэклог (P0-P2)
### P0 (Критично)
- Нет оставшихся P0 задач

### P1 (Важно)
- Страница отдельного товара с детальным описанием и галереей
- Интеграция реальных изображений товаров для всех категорий

### P2 (Желательно)
- Сравнение товаров
- Калькулятор комплекта оборудования для мойки
- SEO-оптимизация (мета-теги, Open Graph)

- Живой чат с менеджером
- Отзывы и кейсы клиентов
